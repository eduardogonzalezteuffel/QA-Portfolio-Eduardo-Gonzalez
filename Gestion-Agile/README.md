# Gestión de Proyectos con Metodologías Ágiles (Scrum)

En esta sección documento mi experiencia y enfoque como **Scrum Master Jr.**, facilitando procesos que aseguran la calidad y la eficiencia del equipo.

## 🎭 Roles y Facilitación
* **Ceremonias:** Conocimiento sólido en la conducción de Daily Stand-ups, Sprint Planning, Reviews y Retrospectivas.
* **Gestión de Impedimentos:** Capacidad para identificar y remover bloqueos que afecten la velocidad del Sprint y el rendimiento del scrum team.
* **Refinamiento:** Colaboración con el Product Owner para asegurar que el Backlog cumpla con el "Definition of Ready" (DoR).

## 📊 Herramientas y Métricas (Jira Software)
* **Configuración de Tableros:** Experiencia configurando versiones y visualizando historias de usuario en el backlog de Jira.
* **Métricas de Equipo:** Seguimiento de Burndown Charts y Velocity para monitorear el progreso del equipo.

## 🛠 Caso Práctico
* **Definition of Done (DoD):** Definición de criterios claros que incluyen pruebas de QA aprobadas y reportes de bugs cerrados antes de terminar el Sprint.
